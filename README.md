# 🎧 Beatify

### A Spotify-Inspired Music Player built with Core Java

---

## 📌 Overview

**Beatify** is a desktop-based music player application inspired by modern streaming platforms. Built entirely using **Core Java (Vanilla Java)**, the project focuses on implementing real-world software design principles including Object-Oriented Programming, event-driven architecture, and file handling.

This project highlights the ability to build a fully functional application without relying on external frameworks.

---

## ✨ Key Features

* 🎵 Seamless audio playback (Play, Pause, Stop)
* ⏭️ Track navigation (Next / Previous)
* 📂 Load and manage local audio files
* 📃 Playlist creation and management
* 🔊 Volume control *(if implemented)*
* 🖥️ Interactive GUI using Swing / JavaFX

---

## 🛠️ Tech Stack

| Technology     | Description                              |
| -------------- | ---------------------------------------- |
| Java           | Core application logic                   |
| Swing / JavaFX | Graphical User Interface                 |
| File Handling  | Managing audio files                     |
| OOP            | Encapsulation, Inheritance, Polymorphism |

## 🚀 Getting Started

### 🔹 Clone the Repository

```bash
git clone https://github.com/harrypotter1808/Beatify.git
```

### 🔹 Run via IDE

1. Open the project in Eclipse / IntelliJ IDEA
2. Locate the main class
3. Run the application

### 🔹 Run using JAR

```bash
java -jar Beatify.jar
```
